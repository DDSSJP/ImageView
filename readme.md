# ImageView for .NET Framework

## 概要

`.NET Framework 4.8`で対応する画像ファイルを表示する画像ビュアーです。

`.bmp .jpg .png .gif`の対応は確認しています。

以下のような特徴があります。

- 枠がないウィンドウで表示
- マウスのクリックとドラッグで簡単に操作
- 細かい拡大、縮小、回転が可能

画像表示に関係するものは全部.NETの機能を使っているのでサンプルプログラムとしても使えます。

## 操作方法

- ウィンドウの端をつかんで上下左右と斜めにウィンドウサイズを変更
- 上の端の内側にコントロールボックスがある
- コントロールボックスはマウスカーソルを重ねると表示される
- コントロールボックスは左から回転、移動、最小化、最大化、閉じる
- そのほかの領域は画像の移動
- 右クリックでコンテキストメニュー

## 使用方法

### コマンドライン

１つ目の引数にファイルパスを指定してください。

`ImageView.exe sample.png`

### エクスプローラー

`ImageView.exe`に画像ファイルを１つドラッグ＆ドロップしてください。

### プログラムから開く

画像ファイルを右クリックして`プログラムから開く(H)`から`ImageView.exe`を選択してください。  
`常にこのアプリを使って.jpgファイルを開く`にチェックをつけておくと以降はImageViewで表示されます。

`プログラムから開く(H)` > `別のプログラムを選択(c)` > `その他のアプリ↓` > `このPCで別のプログラムを探す`
